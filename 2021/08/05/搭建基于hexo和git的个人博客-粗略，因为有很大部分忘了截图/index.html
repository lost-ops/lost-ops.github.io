简单的搭建流程#

准备工作#

首先你需要一个git的账号，很多人都以为github是被墙了，但其实是可以直连的，就是需要多刷新几次。注册完之后需要下载你相应的系统的客户端（比如我就是windows git）然后完成对git的配置，在此不进行详细的叙述，在CSDN的这篇博客中有具体的叙述。

在配置完成后，要进行nodejs的下载安装 我下载的是LTS的，安装完之后可以用

检验是否安装成功，如果出现版本号，安装成功。之后还需要一个markdown语言编辑器，我用的是typora，如果你习惯用记事本也可以。

开始搭建博客#

先在cmd安装cnpm工具，然后安装hexo。

之后可以用hexo -v进行验证，若出现版本号，成功。

打开git客户端(git bash)，用cd命令进入你想要创建blog文件夹的目录，然后mkdir blog建立blog文件夹(若搭建过程中出错可以删了blog重新建立，从头再来)。

hexo初始化blog
